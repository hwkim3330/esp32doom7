DOOM WAD FILE REQUIRED
======================

Place doom1.wad file in this folder.

Download from:
- https://archive.org/details/doom-wad-shareware-1.9
- File size: 4,196,020 bytes
- MD5: f0cefca49926d00903cf57551d901abe

This is the shareware version of DOOM (legal to download).

After placing doom1.wad here:
1. Tools → ESP32 Sketch Data Upload
2. Wait for SPIFFS upload to complete
3. Then upload the main sketch

The data folder should contain:
data/
└── doom1.wad (4.2 MB)
